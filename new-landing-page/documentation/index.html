Installation

Follow the steps below to setup your site template:

Please note that RINJANI is a simple HTML template and not a Wordpress theme/plugin.

  1. Log into your hosting space via an FTP software (Filezilla for example).
  2. Unzip the downloaded package and open the /html folder to find all the template files.
  3. Below is the folder structure and needs to be uploaded to your website root directory:
    • html/assets/css - Stylesheets folder
    • html/assets/fonts – Icon fonts folder
    • html/assets/images - Images folder
    • html/assets/img - Images folder
    • html/assets/js - Javacripts folder
    • html/contact – Contact form PHP files
  4. Upload the HTML files corresponding to the variant of your choice along with all directories in the structure that they appear in your download package. Then, the HTML file should reside in the root directory, and all other files should reside inside their associated folder. You should upload all or specific HTML files as per your need.

Tips Do not start from scratch, use an existing page from the demo and modify it to learn how it works.

Add Custom Color Scheme

The basic process is that we are going to use one of the existing color (color_green.css) and make changes to it to reflect the new custom color scheme. In order to demonstrate, I'm going to create a new custom color scheme based on brown (color_brown.css).

  1. Make a copy of the original color_green.css.


  2. Rename the file (color_green - Copy.css).


  3. Replace color scheme. Using text editor, edit color_brown.css and do a Search/Replace on the file. In my example, I'll change: #0baf4e (green) to #663300 (brown).


  4. Add a link to it in the <head> section of the HTML page.

    <head>
    <-- Theme CSS -->
    <link href="assets/css/theme.css" rel="stylesheet" type="text/css">
    <link href="assets/css/style.css" rel="stylesheet" type="text/css">

    <-- Color Scheme CSS -->
    <link href="assets/css/color_brown.css" rel="stylesheet" type="text/css">

</head>

Background - Video (Vimeo)

Important Notice!
  • The plug in works only if used under a web server. It doesn't work if you run the HTML page as file (file://...).
  • If you want to run it locally on your computer you need a web server installed (for ex: MAMP for Mac, XAMPP for Windows, LAMP for linux) and reach the file from http://localhost... or whatever you mapped in the Host file as localhost.
  • This player doesn't work as background for devices due to the restriction policy adopted by all on managing multimedia files via javascript. It fallsback to the default Vimeo player if used as player (not applied to the body). So video background for mobile device is disabled, only fullscreen poster will be used. Example: data-video-poster="assets/images/bg-home-1.jpg"

  1. Include files:

    <-- CSS -->
<link href="assets/css/jquery.mb.vimeo_player.min.css" rel="stylesheet" type="text/css">

<-- JS -->
<script src="assets/js/jquery.mb.vimeo_player.min.js"></script>

  2. HTML:

    <-- Section - Home Start -->
<section id="home-bg-video" class="bg-cover bg-gray bg-overlay-black-6 h-100 p-0 w-100">
    <-- BG Video - Vimeo -->
    <div id="bgndVideoVimeo" class="player bg-overlay-black-5 h-100 w-100"
        data-property="{videoURL:'https://vimeo.com/108792063'}"
                data-video-poster="assets/images/bg-home-1.jpg">
    </div>
    <-- //.player -->
</section>
<-- //Section - Home End -->

Note You can find this demo in home-background-video-vimeo.html page.

Pie Chart

Setting up Pie Chart is simple. Please use the following codes:

  1. Include files:

    <-- Easy Pie Chart JS -->
<script src="assets/js/jquery.easypiechart.min.js"></script>

<-- Theme JS -->
<script src="assets/js/theme.js"></script>

  2. HTML Code:

    <div class="pie-chart opacity-9-5">
    <div class="chart font-alt text-white" data-percent="0" data-percent-update="83" data-bar-color="#0baf4e" data-track-color="#ffffff">
        <span class="percent font-alt font-w-600 title-extra-large-2">0%</span>
    </div>
    <-- //.chart -->
</div>
<-- //.pie-chart -->

  3. Configuration Parameter:

    You can pass a set of these options to the initialize function to set a custom behaviour and look for the plugin.

    • data-percent-update: The percent number the pie chart should have.
    • data-bar-color: The color of the circular bar.
    • data-track-color: The color of the track for the bar.

Contact Form

Regarding Contact Form as main part of Contact Section, only thing you need to do is to setup your host, username and password. To do this, please open html/contact/send_mail.php file and change this line:

define('SMTP_HOST', 'YOUR_HOSTNAME');
define('SMTP_USERNAME', 'USERNAME@DOMAIN.com');
define('SMTP_PASSWORD', 'YOUR_PASSWORD');
// To address who will receive this email 
$to = 'example@gmail.com';

These settings may be different depending upon your Host Provider. Please contact your Web Server Host for correct details.

Note This will be functional only on your server, php file can not be executed on your local computer.
